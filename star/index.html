星访谈 Star interview

/Part 1 美女玩家的游戏日常

Q : 最开始接触游戏是什么时候？当时都玩些什么？

A : 很小的时候就爱玩各种游戏，当时是那种手柄类型的，最喜欢打超级玛丽这类哈哈。不过那时候就是纯粹爱玩乐。后来大一点喜欢上了当时的魔兽争霸，那时候都偷偷得用光碟玩，都是外文，看不懂哈哈，所以就是乱玩。

Q : 最擅长的是什么游戏？

A : 现在比较喜欢音乐类各种游戏还moba类的，像《自由之战》。音乐类的，像《恋舞OL》、《节奏大师》、《cytus》、《deemo》。 我都好喜欢，感觉歌都非常好。角色的话，一般玩法师辅助比较多，像雷神、法维安这类的。因为这几个英雄是目前版本很强势的中单，我个人喜欢强势点。

/Part 2 萌妹主播初长成

Q : 最早是怎么接触到游戏主播这个行当的？

A : 去年参加EFUN爱游戏全国微竞技大赛<自由之战>项目，没想到一下子拿到了南京赛区的冠军。也是从那个时候开始，慢慢接触到游戏主播这个行当，从此一发不可收拾。第一次解说是EFUN广州赛区的总决赛。当时真的是第一次解说这种现场，还蛮紧张的。不过还好当时男搭档之前认识的，私下也搭档了一下。看着那些精彩的比赛，然后一边去解说，感觉整个人都是沸腾的。

Q : 会自己在家里看一些直播，做一些练习之类的么？

A : 会的，回去看一些，前辈的解说视频，然后自己拿个比赛视频在那练。比如若风、miss这类，原先还会看千年、易水寒，这些前辈们的视频。

/Part 3 人气女神进化论

Q : 不玩游戏的时候会做什么？

A : 会和闺蜜朋友逛逛街、看看电影，节假日还会和闺蜜朋友去旅行。平时还会弹弹钢琴（可惜弹的很烂 哈哈）和吉他（也是半斤八两）。最近每天就直播，再加上今年的一些筹划，还有练练自己的游戏技术。

Q : 目前有另外一半么？没有的话，选择的标准是？

A : 单身哈哈。暂时还没有考虑，不过感觉性格合得来，能照顾好我的，是首选。

Q : 生活中会是一个强势的人么？身边的朋友或者队友是怎么评价你的？

A : 生活中算是个开心果，闺蜜们会觉得我还挺真性情的一个人，可能性格比较活泼，和他们一起就各种逗，队友几乎都当我是个汉子 。

Q : 玩游戏的过程中遇到什么有意思的人或者事情？

A : 游戏让我认识很多朋友，还有一个很重要的闺蜜。我们很多都变成了现实朋友，然后大家在一起爱互相调侃、互损，很有意思。我那个闺蜜和我还不在一个城市，天津的妹子女神一枚，不过经常约假期一起出去旅行。

Q：你最喜欢的解说是？

A：最喜欢若风大大的解说！

Q：前辈们的解说对你有什么影响么？

A：若风大大对战局分析都很好，属于实力解说；我自己的话，其实想把自己往主持解说这一种方向发展；不想单一解说，而且感觉主持人更难，想挑战一下。

Q：未来有什么计划么？

A：未来计划是等毕业后，全身心投入到解说主持直播这块，毕竟找到一份自己喜欢的职业 真的很幸运。希望我在现在选择的这条路上越走越好 。

Q：有什么想对爱游戏的玩家们说的？

A：爱游戏是一个很棒的游戏平台，在这里我认识了很多朋友，也找到了自己喜欢的方向。希望大家也能和我一样找到自己真正喜欢的东西吧~~也希望爱游戏能把EFUN这样的比赛继续办下去，让更多玩游戏的小伙伴有一个展示自己实现梦想的机会。最后还请大家多多关注我的直播呦~
